Лабораторная работа №7

Для предложенного варианта заданий необходимо реализовать веб-сайт в среде Django, удовлетворяющий следующим требованиям:

  1. Количество таблиц в БД - не менее 5. Таблицы должны быть спроектированы в соответствии с тремя нормальными формами
  2. Должны быть реализованы 3 типа связей: one-to-one, one-to-many, many-to-many.
  3. Таблицы создаются с использованием моделей в Django и миграций.
  4. Как минимум, для 1 сущности должны быть реализованы CRUD операции (Create, read, update, delete) + представление списка сущностей.
  5. Остальные таблицы должны быть доступны для редактирования в административной части сайта.
  6. Необходимые пять таблиц должны быть разработаны для моделирования предметной области, заданной вариантом. Необходимо придумать эти таблицы, не забывая про здравый смысл.


Лабораторная работа №8

Необходимо модифицировать лабораторную работу №7:

  1. Добавить средства редактирования для ВСЕХ сущностей.
  2. Добавить минимум 2 роли, соответствующих реальным ролям в выбранной предметной области. Добавить пользователей, обладающих этими ролями.
  3. Реализовать средства регистрации пользователей, аутенфикации.
  4. Добавить средства авторизации и разграничения прав доступа разным ролям к разным сущностям.
